My most productive day

Probably only the hundredth article about AI you've seen today!

Sunday, August 3rd 2025
On Friday I stayed up late to finish outling the plot for my next videogame...
Nothing fancy, just a bullet list of events with a bunch of details, characters, motivations, themes, etc...
I usually have the alarm clock set to 6.30 AM every day, but being so late, I set it to 8 AM.


The following is my Saturday morning:
  1. Woke up at 8 AM, cigarette, bathroom, cigarette, breakfast, cigarette, ...
  2. 8.30 I get on my laptop and dive into my game development endevours.
  3. First thing, the plot sounds cool, but I need feedback on it, so I asked Claude to turn my outline into a proper prose that I can have someone read as a short novel.
  4. After proof reading the thing I sent it to a local TTS model (Kokoro) to get an audio version to share with friends
  5. Go do some laundry
  6. Mow the loan
  7. Come back and, partially thanks to timezones, some friends where online and listened to it right away
  8. We discussed some minor changes, had Claude apply them
  9. Asked Claude to take my initial outline (which splits events into chapters/levels), list of characters, and the prose, and come up with a list of contextualized dialogue lines for each character
  10. Made a Python script to dub them all with Eleven labs
  11. The result was a list of about ~150 mp3 files along the lines of "with clear anxiety in her voice she said bla bla bla"
  12. Damn, it's good, but I only need the bla bla bla part...
  13. Searched for tools to trim audio files quickly, macOS preview allows you to trim videos very easily, but apparently audio is too complicated for them... fine.
  14. Got Claude Code to make me an app where I can drag and drop a folder with mp3 files in it, display them in a list with spectogram, sliders to trim audio start and end, controls to play audio and a save button
  15. Go hung the laundry
  16. Come back... One shotted, that's nice
  17. Spend 30 minutes trimming each audio file
  18. Task Claude Code with allowing attaching audio dubs to dialogue items, assign correct audio files for the game prologue
  19. Go get groceries for a BBQ I have tonight
  20. Come back... One shotted, again
  21. Run some tests
  22. Prepare lunch
  23. Find some bugs, have Claude fix them while I eat
  24. All good, all fixed, lunch done, get coffee and lay down in the sun to relax a bit
  25. Realize it's still 11.30 AM
